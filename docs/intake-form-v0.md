# Intake Form v0

## Purpose

This intake form helps a homeowner prescreen tenant/housemate prospects before scheduling showings.

It is not a formal rental application. It collects structured information to understand timing, lifestyle fit, financial readiness, and application readiness.

## Room Context

- Owner-occupied shared home

- Single-occupancy room
- Private bedroom and private bathroom
- Available for June move-in
- 1st-floor room with high privacy
- Common areas are on upper floors
- Calm, respectful, low-foot-traffic home
