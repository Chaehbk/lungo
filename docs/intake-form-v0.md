## Room Profile Fields

The landlord/homeowner should be able to configure:

- Room title
- Location label
- Available move-in date
- Rent
- Utilities
- Parking
- Other fees
- Private room? yes/no
- Private bathroom? yes/no
- Furnished? yes/no
- Pets allowed? yes/no
- Smoking allowed? yes/no
- Owner-occupied home? yes/no
- Home environment description

Landlords/homeowners should be able to add as many custom other fees as needed.

Examples of other fees:

- Cleaning fee
- Furniture fee
- Storage fee
- Amenity fee
- Parking fee, if not included in parking details
- Other recurring add-on fee

Suggested internal field names:

- `title`
- `location_label`
- `available_move_in_date`
- `rent`
- `utilities_fee`
- `parking`
- `other_fees`
- `has_private_room`
- `has_private_bathroom`
- `is_furnished`
- `pets_allowed`
- `smoking_allowed`
- `is_owner_occupied`
- `home_environment_description`
